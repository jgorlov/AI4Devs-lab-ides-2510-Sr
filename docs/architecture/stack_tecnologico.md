# Stack Tecnológico

Este documento define las tecnologías utilizadas en el proyecto **LTI Antigravity**.

## Frontend
*   **Framework**: React (v18.x)
*   **Lenguaje**: TypeScript
*   **Build Tool**: React Scripts (Create React App)
*   **Estilos**: CSS (Vanilla / CSS Modules)
*   **Internacionalización (i18n)**: `i18next`, `react-i18next`
*   **Testing**: Jest, React Testing Library

## Backend
*   **Runtime**: Node.js
*   **Framework**: Express.js
*   **Lenguaje**: TypeScript
*   **ORM**: Prisma
*   **Base de Datos**: PostgreSQL
*   **Documentación API**: Swagger (swagger-ui-express)
*   **Manejo de Archivos**: Multer

## Infraestructura y Herramientas
*   **Contenedores**: Docker, Docker Compose
*   **Linting/Formatting**: ESLint, Prettier
*   **Control de Versiones**: Git

## Estructura del Proyecto
El proyecto sigue una estructura de monorepo simple con carpetas separadas para `frontend` y `backend`.
